#  Portfolio

A high-performance, responsive portfolio built to showcase software engineering projects. This project emphasizes clean UI/UX, modern CSS architecture, and efficient React patterns.



##  Tech Stack

- **Framework:** [React.js](https://reactjs.org/) (Vite)
- **Styling:** [Tailwind CSS v4.0](https://tailwindcss.com/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Animation:** CSS Keyframes

### 🔹 Dynamic UI Logic
- **Scroll-Aware Navigation:** Implemented a custom `useEffect` hook to monitor `window.scrollY`, dynamically applying glassmorphism effects once a 50px threshold is reached.
- **Particle System:** A procedural background system that generates unique floating elements using `Math.random()`, animated via optimized CSS `drift-slow` keyframes.
